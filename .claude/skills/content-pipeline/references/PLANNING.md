# Stage 2: Content Planning

Create a detailed content strategy plan that prioritizes cultural authenticity over SEO optimization.

## Input

Read `plan/content/{slug}/research.md` from Stage 1. Extract:
- Key findings and verified facts
- Source quality and gaps
- Cultural sensitivity flags
- Community consultation needs

## Planning Approach

1. **Cultural authenticity first** — Authentic representation, community voice, morabeza spirit
2. **SEO optimization second** — Optimize for diaspora discovery after cultural foundation
3. **Multilingual strategy** — English (primary) with adaptation notes for Portuguese/French
4. **Verification planning** — Include accuracy checkpoints and community validation needs

## Content Types

| Type | Focus | Word Count |
|------|-------|------------|
| Heritage Educational Page | Traditions, music, practices, history | 1,400-2,000 |
| Diaspora Connection | Sodade, family heritage, homeland bridge | 1,200-1,600 |
| Directory Entry | Restaurants, hotels, landmarks with context | 400-600 |

## Five-Phase Framework

### Phase 1: Cultural Context Analysis
- Research cultural significance to Brava community
- Identify authentic resident voices and perspectives
- Plan morabeza spirit integration
- Map diaspora connections and homeland links
- Identify verification checkpoints per `docs/10-product/cultural-heritage-verification.md`

### Phase 2: SEO Strategy
- Research diaspora discovery keywords (consult `references/SEO_KEYWORDS.md`)
- Identify cultural term keywords (morabeza, sodade, morna)
- Analyze search intent (informational, connection, planning)
- Plan E-E-A-T signals (experience, expertise, authority, trust)

### Phase 3: Content Outline
For heritage educational pages, target 1,400-2,000 words:
- **Introduction** (100-150 words): Cultural hook, value proposition, primary keyword
- **Cultural Significance** (300-400 words): Community perspective, living tradition
- **Historical Context** (400-500 words): Verified background, evolution, key figures
- **Contemporary Practice** (300-400 words): How tradition lives today, practitioners
- **Diaspora Connection** (200-300 words): Global engagement, sodade, participation
- **Conclusion** (100-150 words): Summary, CTA, morabeza closing

Adapt structure for other content types (directory entries are shorter, diaspora content emphasizes section V).

### Phase 4: Multilingual Adaptation Notes
- **English**: 8th-10th grade reading level, define cultural terms, pronunciation guides
- **Portuguese**: Kriolu integration, Cape Verdean dialect, homeland focus
- **French**: West African connections, francophone diaspora bridges

### Phase 5: Schema & Technical SEO
Plan structured data (Article, Place, Event, FAQ) and meta optimization.

## Cultural Verification Checkpoints

Include in every content plan:
- [ ] Historical claims requiring source verification
- [ ] Cultural practices requiring practitioner validation
- [ ] Family stories requiring community consent
- [ ] Geographic details within Brava Island bounds (lat: 14.80-14.90, lng: -24.75 to -24.65)
- [ ] Cultural term usage verified against `references/GLOSSARY.md`

## Content Plan Output

Save to `plan/content/{slug}/content-plan.md`. Use `references/TEMPLATES.md` for the complete plan template structure.

## Handoff to Stage 3

The content plan should give the Author stage everything needed:
1. Outline with section-by-section breakdown and word counts
2. Key cultural authenticity requirements
3. SEO integration guidance (keywords, E-E-A-T signals)
4. Multilingual adaptation priorities
5. Cultural verification checkpoints

## Checkpoint

Present the content plan to user:
- Content type and word count target
- Section outline with key points per section
- SEO strategy summary
- Cultural verification checkpoints identified
- Multilingual notes

Wait for user confirmation before proceeding to Stage 3 (Author).
