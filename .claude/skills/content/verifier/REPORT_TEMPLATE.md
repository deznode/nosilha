## Template Usage Guidelines

### When to Use This Template

Use this template for every verification report, adapting sections based on verification type:

- **Historical Fact Verification**: Emphasize evidence analysis, source citation, timeline accuracy
- **Cultural Practice Authentication**: Focus on practitioner consultation, living tradition status, community consensus
- **Biographical Verification**: Highlight multiple source validation, family consultation, community memory
- **Bias Assessment**: Emphasize bias detection analysis, colonial perspectives, community voice

### Template Customization

Adapt the template while maintaining core structure:

**Required Sections (Always Include)**:
- Executive Summary with confidence level
- Detailed Verification Findings (use appropriate pattern)
- Bias Detection Analysis
- Source Citation Analysis
- Community Consultation Recommendations
- Corrections Required
- Conclusion with overall recommendation

**Optional Sections (Include When Relevant)**:
- Appendix A: Evidence Trail (for comprehensive verification)
- Appendix B: Community Approval Documentation (when community review complete)
- Additional specialized sections for unique verification needs

### Report Naming Conventions

Save verification reports using consistent naming:

- **Format**: `plan/content/[topic-slug]-verification.md`
- **Examples**:
  - `plan/content/eugenio-tavares-biography-verification.md`
  - `plan/content/traditional-fishing-methods-verification.md`
  - `plan/content/morna-music-heritage-verification.md`
  - `plan/content/independence-history-verification.md`

### Quality Standards

Every verification report should meet these standards:

- **Comprehensive**: Cover all relevant verification aspects
- **Evidenced**: Support all claims with source documentation
- **Community-Centered**: Prioritize community authority and perspective
- **Actionable**: Provide clear recommendations with implementation guidance
- **Professional**: Maintain consistent formatting and structure
- **Transparent**: Document evidence trail and consultation process completely

**Remember**: The verification report serves as both quality assurance documentation and community collaboration record. It protects cultural integrity, supports content creators, and ensures authentic Cape Verdean heritage representation on the Nos Ilha platform.
