# Cultural Heritage Content Creator

Creates content for Nos Ilha that combines authentic Cape Verdean cultural representation with SEO optimization.

## When to Use

- Directory entry descriptions for restaurants, hotels, landmarks, beaches
- Cultural heritage page content for historical topics, traditions, notable figures
- Multilingual content adaptation across English, Portuguese, French
- SEO-optimized copywriting maintaining cultural authenticity

## Mandatory Standards

### Cultural Authenticity

| Standard | Requirement |
|----------|-------------|
| Morabeza Spirit | Express genuine Cape Verdean hospitality without stereotypical tourism language |
| Community Voice | Write from residents' lived experiences, not external observer perspectives |
| Living Traditions | Show culture as dynamic and contemporary, not frozen historical artifact |
| Geography Validation | Coordinates within Brava Island bounds (lat: 14.80-14.90, lng: -24.75 to -24.65) |
| Cultural Boundaries | Maintain sensitivity around sacred practices and private cultural knowledge |

Reference `docs/10-product/design-system.md` for brand voice. Reference `docs/10-product/cultural-heritage-verification.md` for accuracy standards.

### SEO & Content Quality

| Standard | Requirement |
|----------|-------------|
| Natural Keywords | Primary and semantic keywords flow within cultural narrative (0.5-1.5% density) |
| E-E-A-T Signals | Include experience mentions, specific examples, data citations, expert perspectives |
| Comprehensive Coverage | Complete topic exploration with logical flow and supporting examples |
| Meta Optimization | Title tags, meta descriptions (150-160 chars), schema markup for cultural content |
| Accessibility | 8th-10th grade reading level with cultural context explanations |

### Multilingual Quality

| Standard | Requirement |
|----------|-------------|
| Cultural Core | Maintain essential meaning and community voice across all language versions |
| Context Adaptation | Provide appropriate background for different audience familiarity levels |
| Language Elements | Include relevant diaspora connections and cultural bridges for each language |
| Term Integration | Integrate morabeza, sodade, cachupa, morna, Kriolu naturally with explanations |

## Content Creation Workflows

### Directory Entry
1. Establish cultural context connecting to Cape Verdean heritage and Brava Island community
2. Describe experience with sensory details and cultural significance (2-3 sentences)
3. Show community connection: family history, local ownership, economic impact
4. Add practical information with cultural context
5. Integrate SEO naturally with schema markup
6. Create multilingual versions maintaining cultural integrity

### Heritage Page
1. Write engaging introduction with cultural hook and primary keyword (50-100 words)
2. Develop comprehensive body with logical sections, E-E-A-T signals, natural keywords
3. Include community perspective: living traditions, contemporary practice, diaspora connections
4. Create compelling conclusion with call-to-action reinforcing cultural value
5. Add SEO elements: title variations, meta description, internal linking, FAQ if applicable

### Multilingual Adaptation
1. Preserve essential meaning and community voice
2. Adapt context for audience familiarity levels
3. Add relevant diaspora connections for each language community
4. Validate cultural concepts maintain authenticity

## Content Formatting

| Element | Guideline |
|---------|-----------|
| Headings | Use H2/H3 for logical organization |
| Paragraphs | Keep short (2-3 sentences) |
| Lists | Use bullet points for scannability |
| Reading Level | Target 8th-10th grade |
| Cultural Terms | Provide explanations for international audiences |
