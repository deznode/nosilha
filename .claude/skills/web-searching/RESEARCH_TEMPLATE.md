# Research Template

This template shows the expected markdown format for cultural and historical research output.

## Structure

All research files should follow this structure pattern (based on existing cultural research files):

```markdown
# [Clear, Descriptive Title]

## Research Methodology

**Queries Used:**
- Query 1: [exact search query]
- Query 2: [exact search query]
- Query 3: [exact search query]

**Sources Consulted:**
- [Number] academic sources
- [Number] government/official sources
- [Number] community/cultural sources
- [Number] news/media sources

**Timeframe:** [Date range or research period]

## Key Findings

[Narrative synthesis of research findings with inline citations using superscript numbers¹, ², ³]

### [Topic Section 1]

[Detailed findings with direct quotes when important]

> "Direct quote from authoritative source"¹

[More narrative with citations²]

### [Topic Section 2]

[Continue with additional sections as needed]

## Timeline (if applicable)

For historical research, include chronological tables:

| Year | Event | Source |
|------|-------|--------|
| 1462 | [Event description] | ¹ |
| 1680 | [Event description] | ² |
| 1975 | [Event description] | ³ |

## Data Summary

[Structured summaries, statistics, or key data points]

- **Population:** [data with citation]
- **Geographic area:** [data with citation]
- **Cultural significance:** [data with citation]

## Credibility Assessment

**Source Quality:**
- [Assessment of source reliability and authority]
- [Note on consensus vs. contradictions]
- [Identification of gaps in available information]

**Verification Status:**
- ✓ Facts verified across 3+ independent sources
- ⚠ Facts from single source (needs verification)
- ❌ Contradictory information found (details below)

## Contradictions and Gaps

[Note any conflicting information found across sources]

[Identify gaps in available information]

## Works Cited

¹ [Full citation with URL]
[Author/Organization]. "[Title]." [Publication/Website]. [Date]. [URL]

² [Full citation with URL]

³ [Full citation with URL]

[Continue numbered citations for all sources referenced]

## Recommendations for Further Research

- [Follow-up topic 1]
- [Follow-up topic 2]
- [Areas requiring deeper investigation]
```

## Example from Existing Research

Based on existing research files in `plan/content/cultural-research/`, a well-formatted research file includes:

1. **Clear H1 title** describing the research topic
2. **Chronological tables** for timeline-based research (markdown tables with Year, Event, Source columns)
3. **Structured sections** using H2/H3 headings for logical organization
4. **Inline citations** with superscript numbers throughout text (¹, ², ³)
5. **Direct quotes** for important claims with attribution
6. **Works cited section** with numbered references matching inline citations
7. **Rich content** with relevant data, narratives, and detailed information

## File Naming Pattern

Follow these naming conventions:

- `Brava Island [Topic] Research_.md` for general Brava Island topics
- `[Specific Person Name] Biography Research_.md` for notable figures
- `[Cultural Practice] Cultural Research_.md` for traditions and practices
- `[Historical Event] Historical Research_.md` for specific events

Always use title case and include the `.md` extension.
