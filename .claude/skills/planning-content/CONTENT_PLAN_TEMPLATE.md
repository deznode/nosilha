# Content Plan Template

This template provides the complete structure for cultural heritage content plans saved to `plan/content/`.

## File Naming Convention

`plan/content/[topic-slug].md`

**Examples**:
- `plan/content/morna-music-brava-heritage.md`
- `plan/content/eugénio-tavares-poet-legacy.md`
- `plan/content/traditional-cachupa-brava-recipe.md`

## Usage Notes

- **Directory Entries**: Simplify to essential cultural context, SEO keywords, and writing requirements

Always save completed plans to `plan/content/[topic-slug].md` before handing off to content-creator skill.
