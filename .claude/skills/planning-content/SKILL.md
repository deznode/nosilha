# Cultural Heritage Content Planner

This skill should be used when creating detailed content strategy plans for Brava Island cultural heritage educational pages that prioritize Cape Verdean authenticity while optimizing for global diaspora discovery.

## When to Use This Skill

- User requests content planning for Brava Island cultural heritage topics
- Educational page planning needed for traditions, morna music, cultural practices, historical heritage
- Diaspora connection content planning to bridge global Cape Verdean community with ancestral homeland
- Directory entry planning requiring culturally-rich heritage context
- Content strategy requiring cultural verification checkpoints
- Multilingual content planning across English, Portuguese, French

## Content Planning Specialization

This skill focuses on three content types:

1. **Cultural Heritage Educational Pages (80%)**: Plan content about Brava Island traditions, morna music, cultural practices, historical heritage, community stories
2. **Diaspora Connection Content (15%)**: Plan content bridging global Cape Verdean diaspora with ancestral homeland, sodade, family heritage
3. **Directory Entries (5%)**: Plan culturally-rich directory content for restaurants, hotels, landmarks with heritage context

## Planning Approach

Follow this priority order:

1. **Cultural Authenticity First**: Prioritize authentic Cape Verdean representation, community voice, morabeza spirit
2. **SEO Optimization Second**: Optimize for diaspora discovery keywords after cultural foundation established
3. **Multilingual Strategy**: Plan in English (primary) with cultural adaptation notes for Portuguese and French
4. **Verification Planning**: Include cultural accuracy checkpoints and community validation requirements

Reference `docs/DESIGN_SYSTEM.md` for brand voice and `docs/CULTURAL_HERITAGE_VERIFICATION.md` for verification standards.

## Five-Phase Planning Framework

### Phase 1: Cultural Context Analysis

To establish cultural foundation:

1. **Research Cultural Significance**: Determine topic's importance to Brava community and Cape Verdean heritage
2. **Identify Community Perspectives**: Find authentic resident voices and lived experiences to prioritize
3. **Plan Morabeza Integration**: Determine how to embody genuine Cape Verdean hospitality
4. **Map Diaspora Connections**: Identify links to global Cape Verdean diaspora experience
5. **Validate Geography**: Ensure Brava Island context (lat: 14.80-14.90, lng: -24.75 to -24.65)

**Cultural Verification Requirements**:

- Identify historical claims requiring verification per `docs/CULTURAL_HERITAGE_VERIFICATION.md`
- Plan community validation checkpoints for cultural accuracy
- Note sacred practices or cultural boundaries requiring sensitivity
- Determine living tradition vs historical artifact presentation approach

### Phase 2: SEO Strategy Development

To optimize for diaspora discovery:

1. **Research Diaspora Keywords**: Find terms Cape Verdean diaspora uses to discover heritage (e.g., "Cape Verde traditions", "Brava Island culture", "morabeza meaning")
2. **Identify Cultural Term Keywords**: Map secondary keywords (e.g., "morna music", "sodade", "cachupa recipe", "Kriolu language")
3. **Analyze Search Intent**: Determine informational (learning heritage), navigational (finding Brava), transactional (planning travel)
4. **Plan E-E-A-T Signals**: Include first-hand experience mentions, specific community examples, historical citations, expert voices

### Phase 3: Content Outline Creation

To structure comprehensive cultural coverage:

Create detailed outline following this structure for educational pages (target 1,400-2,000 words):

- **Introduction (100-150 words)**: Cultural hook, value proposition, primary keyword, morabeza tone
- **Cultural Significance (300-400 words)**: Community perspective, living tradition, Brava identity, diaspora relevance
- **Historical Context (400-500 words)**: Verified background, evolution, community figures, migration impact
- **Contemporary Practice (300-400 words)**: How tradition lives today, community practitioners, modern adaptations, youth engagement
- **Diaspora Connection (200-300 words)**: Global engagement, sodade, family heritage, participation opportunities
- **Conclusion (100-150 words)**: Cultural value summary, call-to-action, diaspora bridge, morabeza closing

See [CONTENT_PLAN_TEMPLATE.md](CONTENT_PLAN_TEMPLATE.md) for complete template.

### Phase 4: Multilingual Adaptation Strategy

To preserve cultural integrity across languages:

**English (Primary)**:
- Write plan with 8th-10th grade accessibility
- Include cultural term definitions and pronunciation guides
- Plan for international diaspora unfamiliar with Cape Verde

**Portuguese Adaptation Notes**:
- Note Kriolu term integration opportunities
- Plan for Cape Verdean Portuguese dialect considerations
- Identify sections emphasizing homeland community perspective
- Highlight cultural nuances resonating in Portuguese

**French Adaptation Notes**:
- Plan West African cultural connections for francophone diaspora
- Note French-speaking diaspora bridges (France, Luxembourg, Senegal)
- Identify sections emphasizing global diaspora experience
- Include cultural references for French-speaking Cape Verdean community

### Phase 5: Schema Markup & SEO Technical Planning

To enhance search visibility:

1. **Plan Structured Data**: Article, Event (cultural celebrations), Place (Brava locations)
2. **Cultural Entity Markup**: Heritage terms (morabeza, morna, cachupa)
3. **Geographic Coordinates**: Brava Island locations within valid bounds
4. **Meta Optimization**: Title tag variations (50-60 chars), meta description (150-160 chars), internal linking, FAQ schema

See [SEO_KEYWORD_GUIDE.md](SEO_KEYWORD_GUIDE.md) for diaspora discovery keyword research patterns.

## Cultural Heritage Planning Principles

**Remember these non-negotiable principles**:

1. **Authenticity Over Optimization**: If cultural authenticity and SEO conflict, always prioritize authentic Cape Verdean representation
2. **Living Tradition Focus**: Present culture as dynamic and contemporary, not frozen historical artifact
3. **Community Benefit**: Plan content serving Brava Island community interests and economic benefit
4. **Diaspora Bridge**: Every plan should facilitate meaningful connections between diaspora and ancestral homeland
5. **Cultural Boundaries**: Respect sacred practices, private knowledge, and require community consent

Every content plan preserves irreplaceable Cape Verdean traditions while honoring community voice and creating authentic bridges between Brava Island heritage and the global Cape Verdean diaspora.
