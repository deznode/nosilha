# Research Command

Conduct comprehensive cultural and historical research using advanced web search techniques, with findings automatically saved as structured markdown.

## Usage

```bash
/research <research-topic>
```

### Examples

```bash
/research Brava Island volcanic geology and formation history
/research Cape Verdean emigration patterns to New England 1900-1920
/research Eugénio Tavares poetry and the evolution of morna music
/research Traditional Cape Verdean fishing practices on Brava Island
/research Portuguese colonial administration in Cape Verde 1800s
```

## What This Command Does

Performs comprehensive web research on the specified topic:

- Searches multiple authoritative sources (academic, historical, cultural)
- Cross-references facts for accuracy and verification
- Synthesizes findings into a structured markdown report
- Saves research to `plan/content/` with automatic directory classification
- Provides summary with file path, key findings, and source quality assessment

## Implementation

Conduct comprehensive cultural and historical research on the following topic:

**$ARGUMENTS**

Research should:
- Use multiple authoritative sources for verification
- Save findings as structured markdown in `plan/content/` following existing patterns
- Include chronological tables for timeline-based research
- Provide inline citations with complete "Works cited" section
- Focus on cultural heritage topics relevant to Nos Ilha platform (Brava Island, Cape Verde, diaspora history, cultural figures, traditional practices)

After completing research, provide a summary including:
- File path where research was saved
- Directory placement reasoning
- 2-3 key findings
- Source quality assessment
- Recommendations for further research

## Tips for Better Results

**Good topics** (specific and focused):
- "Brava Island volcanic formation and geological history"
- "Cape Verdean whaling industry connections to New England 1800s"
- "Eugénio Tavares' influence on morna music evolution"

**Too broad** (add specifics):
- "Cape Verde history" → Include time period or specific aspect
- "Brava Island" → Focus on geography, culture, history, etc.
- "Migration" → Specify origin, destination, time period

**Tips**:
- Include time periods for historical research
- Specify geographic locations when relevant
- Name specific people, events, or cultural practices
- Combine related topics for comprehensive coverage
